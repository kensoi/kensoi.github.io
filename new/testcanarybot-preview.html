kensoi/testcanarybot - асинхронный фреймворк для создания чатботов ВКонтакте

Преимущества:

  1. Система обработчиков, подключаемых через модули
  2. Полезный инструментарий
  3. Блокировка обработчика до получения следующего сообщения

Установка:

По стандарту testcanarybot можно установить через PIP. Для этого у вас должен быть доступ к cmd и pip, чтобы ввести одну из этих команд:
Pypi [нажмите]

"pip install [--upgrade] testcanarybot"

Способ через Github [нажмите]

Stable

"pip install [--upgrade] https://github.com/kensoi/testcanarybot/tarball/stable"

Unstable

"pip install [--upgrade] https://github.com/kensoi/testcanarybot/tarball/unstable"

Dev

"pip install [--upgrade] https://github.com/kensoi/testcanarybot/tarball/unstable"
